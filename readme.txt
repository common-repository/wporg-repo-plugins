== Description ==

A widget to display plugins listed in the WordPress.org repository by author.

== Installation ==

1. Upload plugin files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. On the 'Widgets' page, drag the WPORG Plugins Repo widget to a sidebar and configure. Only requirement is wp.org username.
